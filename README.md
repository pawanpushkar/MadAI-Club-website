🌐 MadAI Club Website

This repository contains the official website for the MadAI Club, designed to showcase the club’s activities, events, and initiatives related to Artificial Intelligence and technology.

📌 Project Overview

The MadAI Club Website is a modern and responsive web platform built to:

Present information about the club
Highlight events, workshops, and projects
Provide a user-friendly interface for visitors
Promote AI awareness and community engagement
🚀 Features
🎨 Responsive and clean UI design
📱 Mobile-friendly layout
🧠 AI club information and vision
📅 Events and activities section
📩 Contact / Join section
🛠️ Technologies Used
HTML5
CSS3
JavaScript

⚙️ Installation & Setup

Clone the repository:

git clone https://github.com/pawanpushkar/MadAI-Club-website.git

Navigate to the project folder:

cd MadAI-Club-website
Open index.html in your browser.
